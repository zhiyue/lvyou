###“短信指令:restart - 控制开发板重启”使用说明
####指令的作用
　　给开发板中的任意一张SIM卡发送 *#restart# 指令，可以控制开发板立即重启。
####指令的格式（示例）
```
*#restart#000000
```
####指令参数说明
 | 参数（示例） |说明|
| ------------  |--|
|*#restart#   |短信指令名称|
|000000 |用户密码（不能少于4个字符）|

####指令反馈结果
成功时开发板回复短信：
```
设备498060912345接到了您的重启指令。
```

失败时开发板回复短信：
```
用户密码不正确！
```

####指令反馈结果说明
无
