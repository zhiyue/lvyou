###“短信指令:pwd - 修改用户密码”使用说明
####指令的作用
　　给开发板中的任意一张SIM卡发送 *#pwd# 指令，可以修改用户密码。
####指令的格式（示例）
```
*#pwd#000000#123456
```
####指令参数说明
 | 参数（示例） |说明|
| ------------  |--|
|*#pwd#   |短信指令名称|
|000000 |旧用户密码（不能少于4个字符）|
|123456 |变更后的新用户密码（不能少于4个字符）|

####指令反馈结果
成功时开发板回复短信：
```
设备 498060912345 的用户密码已经变更成功。
```

失败时开发板回复短信：
```
设备 498060912345 的用户密码变更失败！
```

####指令反馈结果说明
无
